# GOG Guru

- Enter budget and games/tags you like.
- See games you might like

## Implementation

### Prerequisites

- List of GOG game [IDs][1]
- GOG API [documentation][2] (slightly outdated)

### Custom tools

- `mkdb`: creates a game database using the GOG API and game ID list

## Difficulties

  match docs
